# 應用程式與文件的翻譯狀態

狀態每小時更新一次，僅反映 Crowdin 上的狀態。

:::{提示}

請注意，校對百分比必須低於翻譯百分比，但他們決定將其顯示在翻譯百分比之上。 最好只查看翻譯百分比來判斷是否還有工作需要完成。

:::

:::{admonition} 最終發布所需時間
:class: note

在 Crowdin 完成翻譯後，每天都會進行同步到 Github 的排程，將翻譯作為 pull request（PR）傳送到 Github。

PR 的接受是一項手動任務，將儘快完成。

:::
