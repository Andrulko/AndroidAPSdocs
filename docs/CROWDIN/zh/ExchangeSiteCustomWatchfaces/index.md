# 自訂錶盤交換站

在此您可以下載其它使用者與您分享的自訂錶盤 Zip 檔案。

這些 Zip 檔案可以像往常一樣透過 Pull Request 上傳至 Github 中的 "ExchangeSiteCustomWatchfaces" 資料夾。

在合併 Pull Request 的過程中，文件團隊將提取 CustomWatchface.png 檔案，並在其前加上 Zip 檔案的檔名作為前綴。
